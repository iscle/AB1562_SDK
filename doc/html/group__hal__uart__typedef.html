typedef void(* hal_uart_callback_t) (hal_uart_callback_event_t event, void *user_data)
 This typedef defines user's callback function prototype. This callback function will be called in UART interrupt handler when UART interrupt is raised. User should call hal_uart_register_callback() to register callbacks to UART driver explicitly. Note, that the callback function is not appropriate for time-consuming operations.
parameter "event" : for more information, please refer to description of hal_uart_callback_event_t. parameter "user_data" : a user defined data used in the callback function.
